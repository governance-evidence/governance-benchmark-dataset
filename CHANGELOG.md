## [0.1.0] - 2026-03-27

Initial public release. Zenodo: [10.5281/zenodo.19248723](https://doi.org/10.5281/zenodo.19248723).

### Added

- Curated benchmark dataset covering four decision system architectures:
  deterministic rule engines, hybrid ML + rules, streaming feature-driven,
  and agentic AI.
- Evaluation harness for cross-architecture governance evidence feasibility.
- Governance evidence feasibility matrix across N4 framework properties.
- Apache-2.0 license for code, CC BY 4.0 for dataset content.
- CITATION.cff for academic citation.
